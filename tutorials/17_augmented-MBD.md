# Augmented Multibody Dynamic (MBD) approach

[![Augmented Multibody Dynamic (MBD) approach](https://img.youtube.com/vi/m9jFbnRjKRw/0.jpg)](https://www.youtube.com/watch?v=m9jFbnRjKRw "Augmented Multibody Dynamic MBD approach - click to watch")

In this video, we create the full expanded _augmented technique_ for MBD
simulations. This is the full form that most dynamic simulation
softwares assemble in the background after you create parts, assign
constraints, and apply forces. 


_To run this tutorial locally, download [this file](/tutorials/17\_augmented-MBD.jl) and open it with
[Pluto.jl](https://plutojl.org)._
